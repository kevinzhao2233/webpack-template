使用图片

webpack 4 时通常使用 copy-webpack-plugin 插件拷贝资源到目标文件夹。

webpack 5 内置了对资源的处理方式，配置时在 module.rules 中设置图片的 type = 'asset/resource'。

再配合 html-loader 和 css-loader 就可以在 HTML 和 CSS 文件中使用图片了。

SCSS

对于 SCSS 文件的处理，需要经过 sass-loader、postcss-loader、css-loader、style-loader， 作用分别是将 SCSS 转化为 CSS、CSS 后处理（如进行 autoprefixer 处理等）、CSS 文件处理（如处理CSS 文件中的图片路径等）、 最后将生成的 CSS 代码放到 HTML 文件的 style 标签中

具体使用效果可以在项目中查看。

Babel

实际配置 Webpack 时只需要一个 babel-loader 即可，更多的是配置 .babelrc 文件，或者添加额外的 babel 插件。

babel 7 开始，很多配置都减少了，比如预设，只需要配置 "presets": ["@babel/preset-env"]， 再通过插件添加一些正在提案或者还未加入到 @babel/preset-env 中的功能即可。

具体使用效果在项目中查看。
